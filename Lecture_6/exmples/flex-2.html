شرح خاصية flex للأبناء في Flexbox

مثال 1: flex: 1 (تقسيم متساوي)

جميع العناصر تأخذ نفس المساحة المتبقية في السطر (كل عنصر يكبر أو يصغر بالتساوي حسب حجم الحاوية).
A
B
C
.box { flex: 1; }

مثال 2: flex: 2 و flex: 1 (توزيع غير متساوي)

العنصر الأول يأخذ ضعف المساحة لكل من العنصرين الآخرين.
(النسب: A=2، B=1، C=1)
A
B
C
.box:first-child { flex: 2; }
.box:not(:first-child) { flex: 1; }

مثال 3: flex: 1 0 (ينمو ولا ينكمش)

العنصر ينمو ليملأ الفراغ لكن لا ينكمش إذا صغرت الحاوية.
لاحظ كيف قد يخرج العنصر عن الحاوية إذا لم يكن هناك مساحة كافية.
A
B
C
.box:first-child { flex: 1 0; }
.box:not(:first-child) { flex: 0 1 60px; }

مثال 4: flex: 0 1 (ينكمش ولا ينمو)

العنصر يبقى بحجمه الأساسي، ينكمش إذا صغرت الحاوية، لكنه لا ينمو أبداً ليملأ الفراغ.
A
B
C
.box { flex: 0 1 100px; }

مثال 5: flex: 0 0 auto (حجم ثابت)

العنصر يبقى دائماً على حجمه الأصلي، لا ينمو ولا ينكمش مهما كان حجم الحاوية.
(مفيد لعناصر مثل الأزرار أو الصور التي لا تريدها أن تتغير أبداً)
A
B
C
.box { flex: 0 0 auto; width: 90px; }
