شرح display في CSS - أمثلة أساسية

1️⃣ display: block

هذا عنصر block
يأخذ كامل عرض السطر ويبدأ بسطر جديد.
← مثال عملي: <div> أو <p>

2️⃣ display: inline

عنصر Inline 1 عنصر Inline 2 عنصر Inline 3
← كل عنصر يظهر بجانب الآخر مباشرة ولا يبدأ من سطر جديد.
<span> أو <a>

3️⃣ display: inline-block

Inline-Block 1 Inline-Block 2 Inline-Block 3
← تظهر بجانب بعضها (مثل inline)،
ويمكن تحديد العرض والارتفاع (مثل block).
مفيد للأزرار والقوائم الأفقية.

4️⃣ display: flex

Flex Item 1
Flex Item 2
← كل عنصر (flex item) يتوزع تلقائيًا بشكل مرن داخل الحاوية.
مفيد جداً لبناء layouts احترافية.

5️⃣ display: none

هذا العنصر ظاهر الآن.
هذا العنصر مخفي باستخدام display: none.
← العناصر التي تستخدم display: none لا تظهر أبداً ولا تشغل أي مساحة في الصفحة.
