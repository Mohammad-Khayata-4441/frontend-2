شرح موسّع لبعض خصائص CSS مع أمثلة تطبيقية

1️⃣ الخلفية background

background-color

تستخدم لإعطاء لون خلفية لأي عنصر.

مثال: background-color: #ffe3e3;
div {
          background-color: #ffe3e3;
        }

background-image

تُستخدم لإضافة صورة كخلفية للعنصر. يمكنك تكرار الصورة أو منع التكرار.

مثال: صورة خلفية
div {
          background-image: url('bg.jpg');
          background-repeat: no-repeat;
          background-size: cover;
        }

background-gradient

إنشاء تدرج لوني كخلفية (خطي أو دائري).

تدرج لوني خطي
div {
          background: linear-gradient(90deg, #36d1c4, #5b86e5);
        }

background-attachment

هل تبقى الخلفية ثابتة أم تتحرك مع التمرير؟

خلفية ثابتة عند التمرير (fixed)
div {
          background-attachment: fixed;
        }

background-position

تحديد مكان ظهور صورة الخلفية داخل العنصر.

خلفية يمين-أسفل 
div {
          background-position: right bottom;
        }

background-size

التحكم بحجم صورة الخلفية.

contain / 40px 
div {
          background-size: 40px auto;
        }

2️⃣ مؤشر الفأرة cursor

تغيير شكل مؤشر الفأرة عند المرور فوق عنصر.

القيمة الوصف مثال عملي
pointer يد، غالبًا للروابط والأزرار pointer
default السهم الافتراضي default
text مؤشر تحرير النصوص text
move سهم النقل move
not-allowed علامة المنع not-allowed 
button {
          cursor: pointer;
        }

3️⃣ خصائص القوائم list-style

list-style-type

تغيير شكل نقاط/أرقام القوائم.

  • HTML
  • CSS
  • JavaScript
ul {
          list-style-type: square;
          list-style-position: inside;
        }

list-style-image

وضع صورة كرمز للقائمة بدل الشكل الافتراضي.

  • مثال صورة 1
  • مثال صورة 2
ul {
          list-style-image: url('icon.png');
        }

4️⃣ ::before و ::after

pseudo-elements تُستخدم لإضافة محتوى قبل أو بعد العنصر دون تعديل HTML، ويمكنك تزيينها كما تريد.

هذا نص تجريبي 
.before-after::before {
          content: "⭐ ";
          color: #ffd600;
        }
        .before-after::after {
          content: " 🎯";
          color: #6dd47e;
        }

5️⃣ الفرق بين outline و border

border outline
Border 
.border-demo {
          border: 2px solid #1976d2;
        }
 Outline 
.outline-demo {
          outline: 2px solid #d7263d;
        }
  • border: يؤثر على أبعاد العنصر، ويمكن تحديد كل جانب.
  • outline: إطار خارجي لا يؤثر على الأبعاد ولا يمكن تحديد كل جانب منفصل.

6️⃣ text-transform و text-decoration

text-transform

uppercase: كل الحروف كبيرة
LOWERCASE: كل الحروف صغيرة
capitalize: أول حرف من كل كلمة كبير
p {
          text-transform: uppercase | lowercase | capitalize;
        }

text-decoration

underline: خط تحت النص
overline: خط فوق النص
line-through: خط وسط النص
p {
          text-decoration: underline | overline | line-through | none;
        }

7️⃣ box-sizing و الصندوق box model

ما هو الصندوق في CSS؟

  • content: المحتوى نفسه (نص/صورة...)
  • padding: المسافة الداخلية حول المحتوى
  • border: إطار يحيط بالبادينج
  • margin: المسافة الخارجية خارج الإطار

        +------------------------------+
        |           margin             |
        |  +------------------------+  |
        |  |        border          |  |
        |  |  +------------------+  |  |
        |  |  |    padding       |  |  |
        |  |  |  +-----------+   |  |  |
        |  |  |  | content   |   |  |  |
        |  |  |  +-----------+   |  |  |
        |  |  +------------------+  |  |
        |  +------------------------+  |
        +------------------------------+

box-sizing

  • content-box: (الوضع الافتراضي) - width/height = للمحتوى فقط.
  • border-box: width/height تشمل البادينج والبوردر.
< class="example flex-row">
content-box
.box1 {
          width: 200px;
          padding: 20px;
          border: 4px solid #5c51ff;
          box-sizing: content-box;
        }
border-box
.box2 {
          width: 200px;
          padding: 20px;
          border: 4px solid #5c51ff;
          box-sizing: border-box;
        }
